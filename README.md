## 🐧 Linux 終端機常用指令大權

### 🛠️ 系統管理與資訊 (System Management)

用於掌握主機狀態、電源控制及網路設定。

| 指令 | 用途說明 |
| --- | --- |
| `sudo poweroff` | 立即關機 |
| `sudo reboot` | 重新啟動 |
| `hostname` | 顯示目前的主機名稱 |
| `uname -a` | **[新增]** 顯示系統內核版本與硬體架構資訊 |
| `top` / `htop` | **[新增]** 即時監看系統資源（CPU/RAM）與進程 |
| `df -h` | **[新增]** 查看磁碟剩餘空間（以易讀格式顯示） |
| `ip a` | 查看所有網路介面的 IP 與 MAC 位址 |
| `ping <網址/IP>` | **[新增]** 測試網路連線是否通暢 |

### 📦 套件管理 (APT Package Manager)

適用於 Debian/Ubuntu 系列的軟體安裝與更新。

* **更新清單：** `sudo apt update` (同步遠端伺服器的軟體庫資訊)
* **升級軟體：** `sudo apt upgrade` (更新已安裝的軟體)
* **全局升級：** `sudo apt full-upgrade` (自動處理相依性變動的升級)
* **安裝軟體：** `sudo apt install <套件名稱>`
* **移除軟體：** `sudo apt remove <套件名稱>` **[新增]**
* **清理冗餘：** `sudo apt autoremove` **[新增]** (移除不再需要的依賴套件)

---

### ⚙️ 服務控制 (Systemd)

用於管理系統後台服務（如 Nginx, Docker, MySQL 等）。

* **載入變更：** `sudo systemctl daemon-reload`
* **開關服務：** `start` (啟動) / `stop` (關閉) / `restart` (重啟)
* **狀態檢查：** `sudo systemctl status <服務>`
* **自啟設定：** `enable` (開機自啟) / `disable` (取消自啟)
* **日誌查看：** `journalctl -u <服務>` **[新增]** (查看特定服務的錯誤日誌)

### 📂 檔案與目錄操作 (File & Directory)

最頻繁使用的基礎路徑切換與檔案管理。

#### 1. 瀏覽與定位

* `pwd`：顯示當前路徑 (Print Working Directory)
* `ls`：列出文件（常用組合：`ls -al` 列出所有隱藏檔與詳細資訊）
* `cd <路徑>`：切換目錄（`cd ..` 上一層、`cd ~` 回家目錄）

#### 2. 建立、複製與移動

* `mkdir -p <路徑>`：遞迴建立多層資料夾
* `touch <檔名>`：建立空白文件或更新文件時間戳
* `cp -r <來源> <目的>`：複製檔案或資料夾 (`-r` 用於資料夾遞迴)
* `mv <來源> <目的>`：移動檔案或重新命名

#### 3. 刪除

* `rm <檔案>`：刪除檔案
* `rm -rf <資料夾>`：**強制遞迴刪除**（請謹慎使用）
* `rmdir`：刪除空目錄

### 🔍 內容查看與過濾 (Text Processing) **[新增]**

這部分是您原始列表中較缺少的，但在排錯時非常重要。

| 指令 | 用途說明 |
| --- | --- |
| `cat <檔案>` | 直接在終端機列印出完整文件內容 |
| `less <檔案>` | 分頁查看大型文件（按 Q 退出） |
| `head` / `tail` | 查看文件的前/後 10 行（`tail -f` 可即時監控日誌更新） |
| `grep "關鍵字" <檔案>` | 在文件中搜尋特定字串 |
| `find <路徑> -name <名稱>` | 在指定目錄下搜尋檔案 |

---

### 🔐 權限管理 (Permissions) **[新增]**

控制誰可以讀取或修改檔案。

* **更改權限：** `chmod <權限數值> <檔案>` (例如 `chmod 755 script.sh`)
* **更改擁有者：** `sudo chown <用戶>:<群組> <檔案>`
* **切換用戶：** `sudo -i` (切換為 root 身份)

---

### 💡 小撇步

1. **Tab 鍵：** 輸入指令或路徑到一半時按 Tab 可自動補齊。
2. **Ctrl + C：** 強制中斷目前正在執行的程式。
3. **history：** 查看過去輸入過的指令紀錄。
